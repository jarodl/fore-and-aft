fore-and-aft
============

Solves fore and aft puzzles using DFS and BFS.

Running
-------

    make
    ./fore-and-aft

Testing
-------

From the root directory:

    cd tests/
    make
    ./fore-and-aft-tests

In order to run the tests you need CppUTest. You can grab CppUTest from their
website (or use the version included with this repo).
